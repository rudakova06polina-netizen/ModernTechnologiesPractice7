# Часть 4: Thymeleaf

1. Что делает `th:object`?

Ответ: th:object привязывает форму к определённому объекту из модели.

2. Как `th:field="*{name}"` связывает поле формы с объектом?

Ответ: Звёздочка *{name} ссылается на объект, который указан в th:object выше. Thymeleaf ищет в том объекте поле name и вызывает геттер, чтобы подставить текущее значение в поле.

3. Что означает `redirect:` в возвращаемой строке контроллера?

Ответ: Он предотвращает повторную отправку формы при обновлении страницы. Без redirect: сервер просто отдал бы HTML-страницу, и при обновлении пользователь отправил бы данные ещё раз.
